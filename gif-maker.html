❓ Frequently Asked Questions – GIF Maker

How many images can I add?
You can add as many images as you like. However, for smooth performance keep it under 30 frames. Each frame adds to the final file size and processing time. For long animations (50+ frames), a desktop tool like GIMP or Photoshop will be more efficient.
What is frame delay?
Frame delay is the time (in milliseconds) each image is shown before moving to the next. 100ms = 10 frames per second (smooth animation). 200ms = 5fps (standard). 500ms = 2fps (slow slideshow). Lower delay = faster animation = larger file size.
Why is my GIF large in file size?
GIF file size depends on the number of frames, image dimensions, and color complexity. To reduce file size: lower the output width (e.g. 320px), increase frame delay (e.g. 300ms), or reduce the number of frames. The Quality setting (lower = higher quality but larger) also affects size.
Can I reorder frames after uploading?
The frames are displayed in upload order. You can remove individual frames using the × button and re-upload them in the correct order. For precise reordering, rename your image files (01.jpg, 02.jpg, etc.) before uploading so they load in the right sequence.
