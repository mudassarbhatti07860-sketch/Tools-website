❓ Frequently Asked Questions – PDF Splitter

Is this PDF splitter safe? Is my file uploaded anywhere?
Your PDF file is never uploaded to any server. This tool works entirely in your browser using PDF.js for parsing and a custom JavaScript PDF writer for generating split files. No data leaves your device, making it completely safe for confidential documents, legal contracts, medical records and personal files.
What are the different split modes?
All Pages: Every page becomes a separate PDF file. Custom Ranges: You define which pages go into each output file (e.g. 1-5, 6-10, 11-15). Fixed Interval: The PDF is split every N pages (e.g. every 3 pages creates a new file). Visual Select: You click on page thumbnails to hand-pick exactly which pages to extract into a single PDF.
Can I download all split files at once?
Yes! After splitting, you can click "Download All as ZIP" to download all output PDF files in a single ZIP archive. You can also download each file individually by clicking the download button next to each output file in the results list.
How does the Custom Ranges mode work?
In Custom Ranges mode, you add one or more range entries. Each entry becomes one output PDF file. You can specify ranges like "1-5" (pages 1 to 5), "3,7,9" (specific pages), or "1-3,8,10-15" (mix of ranges and pages). Each range row produces a separate PDF file named with your chosen prefix and a number.
Is there a page or file size limit?
This tool supports PDF files up to 200MB. There is no limit on the number of pages — PDFs with hundreds of pages work fine. However, very large files may take longer to process since all computation happens in your browser. For extremely large documents (500+ pages), processing may take a minute or more.
