🛡️ Password Security Tips
📏
Use 16+ Characters
Length is the most important factor. A 16-char password is astronomically harder to crack than 8 chars.
🔀
Mix All Character Types
Combine uppercase, lowercase, numbers and symbols to exponentially increase the keyspace.
🚫
Never Reuse Passwords
Use a unique password for every account. A breach on one site won't compromise your others.
🗝️
Use a Password Manager
Store your passwords in a trusted manager like Bitwarden or 1Password — never in plain text.
📲
Enable 2FA
Two-factor authentication adds a second layer of security even if your password is compromised.
🔄
Change Periodically
Rotate passwords for critical accounts (banking, email) every 3–6 months to limit exposure.

A password is a random string of characters. A passphrase is a sequence of random words (e.g. "correct-horse-battery-staple"). Passphrases are often longer, easier to remember, and just as secure or more so than shorter character-based passwords.
Entropy (measured in bits) quantifies how unpredictable a password is. More bits = harder to crack. 40 bits is weak, 60–80 bits is good, and 100+ bits is considered very strong for most use cases.
